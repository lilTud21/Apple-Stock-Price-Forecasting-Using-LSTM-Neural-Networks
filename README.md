# Apple-Stock-Price-Forecasting-Using-LSTM-Neural-Networks

Developed a time-series forecasting model to predict Apple Inc.'s stock prices using Long Short-Term Memory (LSTM) neural networks. Employed data preprocessing techniques, including normalization and feature engineering, to optimize model performance. Integrated technical analysis indicators to enhance prediction accuracy, and evaluated results using metrics like MAE and RMSE. Demonstrated advanced proficiency in Python, data visualization, and deep learning frameworks (TensorFlow/Keras) for financial modeling
